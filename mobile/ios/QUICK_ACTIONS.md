# Быстрые ярлыки (iOS Home Screen Quick Actions)

## Возможности
- Нативные быстрые ярлыки на iOS (iOS 14+), до 4 штук.
- Динамическая генерация ярлыков в рантайме.
- Кастомизация: название, иконка (SF Symbols), действие при запуске.
- Обработка запуска приложения через ярлык с передачей параметров.
- Локализация на русский язык.

## Структура проекта
```
mobile/ios/WishShareMobile/
  App/
    AppDelegate.swift
    SceneDelegate.swift
    AppState.swift
    WishShareMobileApp.swift
  QuickActions/
    QuickAction.swift
    QuickActionManager.swift
  Views/
    MainView.swift
    FeatureView.swift
    SettingsView.swift
  Resources/
    Info.plist
    ru.lproj/Localizable.strings
```

## Сборка
1. Откройте `mobile/ios/WishShareMobile` в Xcode и создайте iOS App target с именем `WishShareMobile`.
2. Установите `Info.plist` из каталога `Resources` для таргета.
3. Включите поддержку сцен (UIScene) и URL‑схему `wishshare` (есть в Info.plist).
4. Убедитесь, что версия iOS — 14.0 или выше.

## Использование
- Запустите приложение: на главном экране (MainView) доступен раздел «Ярлыки».
- В «Настройках ярлыков» можно:
  - Включать/выключать быстрые ярлыки.
  - Добавлять новый ярлык, менять название, иконку и действие.
  - Удалять ярлыки.
- Ярлыки появляются при долгом тапе по иконке приложения на домашнем экране.

## Обработка запуска через ярлык
- Реализована в `AppDelegate` и `SceneDelegate` через `QuickActionManager.handle(...)`.
- При запуске приложение открывает нужный экран (`FeatureView`) согласно выбранному действию.

## Тестирование
- Минимальная поддерживаемая версия iOS: 14.0.
- Проверьте сценарии:
  1. Первичный запуск без ярлыков (дефолтные добавляются).
  2. Включение/отключение ярлыков.
  3. Создание нового ярлыка, смена иконки и названия.
  4. Запуск из ярлыка при закрытом приложении.
  5. Запуск из ярлыка при свёрнутом приложении.
  6. Обновление iOS — ярлыки остаются валидными.

## Локализация
- Все пользовательские строки находятся в `Resources/ru.lproj/Localizable.strings`.

## Ограничения
- iOS отображает максимум 4 быстрых ярлыка.
- Для собственных иконок нужны ассеты; по умолчанию используются SF Symbols.
